# Acura RDX Engine Installation Guide

An interactive, full-page website for viewing the 65-step engine installation procedure for a 2007 Acura RDX L4-2.3L Turbo.

## Features

- **Full-page step display**: Each step takes up the entire viewport for easy reading
- **Arrow key navigation**: Use ← → arrow keys to navigate between steps
- **Image support**: Displays relevant images from the original PDF for each step
- **Progress indicator**: Visual progress bar at the top showing completion status
- **Responsive design**: Works on desktop and mobile devices
- **Smooth transitions**: Fade-in animations when switching between steps

## Usage

1. Open `index.html` in a web browser
2. Use the arrow keys (← →) or the navigation buttons at the bottom to move between steps
3. Each step displays:
   - Step number
   - Complete step instructions
   - Relevant images from the original manual

## Files

- `index.html` - Main website file (contains all steps and functionality)
- `images/` - Directory containing extracted images from the PDF
- `Engine Installation-AcuraRDX.pdf` - Original PDF manual

## Navigation

- **Left Arrow / Up Arrow**: Previous step
- **Right Arrow / Down Arrow**: Next step
- **Navigation Buttons**: Click the ← → buttons at the bottom

## Technical Details

- Pure HTML, CSS, and JavaScript (no dependencies)
- All step data is embedded in the HTML file
- Images are loaded from the `images/` directory
- Works offline once all files are downloaded
